# Panel ADM

Carpeta reservada para la documentación, diseño y desarrollo del panel administrativo profesional de Kingdom GPS.

Aquí se organizarán progresivamente:

- Editor de objetos e inventario.
- Editor de mundo y mapa.
- Biblioteca y generador de iconos.
- Vista integrada del juego.
- Gestor de sistemas y versiones.
- Sincronización local/servidor.
- Publicación, pruebas y rollback.
- Ideas, roadmap e informes técnicos.

> Regla: los cambios se documentarán y probarán antes de publicarse en producción.
